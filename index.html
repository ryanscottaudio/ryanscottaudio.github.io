Side projects

cpf-class-signer-upper

Web automation tool written in Typescript and running on AWS Lambda

Problem: the classes my girlfriend and I want to take at our gym have very few slots and fill up extremely quickly. Worse, different classes become available at different times. How are we ever going to get swole?
Solution: Write a script that uses headless Chrome to check the site every 15 minutes and sign up the minute the class becomes available.

  • Grabs classes we want to attend and their times from a Google Sheet using a Node.js wrapper around the Sheets API
  • Uses Playwright and headless Chrome to log into each of our accounts and attempt to sign up for each class
  • Sends an email to us using Nodemailer when a class is successfully signed up for
  • Includes robust error states to recover from errors and logging to ensure I know what's going on
  • Runs every 15 minutes on an AWS Lambda running Node.js
  • Is automatically built with Rollup and deployed to AWS Lambda whenever a change is pushed/merged to the main branch using GitHub Actions

quarantine-fishbowl

Fishbowl game built with React, Node.js, and Socket.IO

Back in the 2020 quarantine, I wanted to create a way for my friends and family to connect that felt familiar and fun. To that end, I decided to build a web version of the popular game Fishbowl that we could all play while in Zoom sessions together.

  • Built using React with a Node.js back-end
  • Uses React Router for navigation
  • Connects client and server for real-time updates in both directions using Socket.IO
  • Allows for multiple running games at once using room IDs
  • Deployed to Heroku and can be played here
